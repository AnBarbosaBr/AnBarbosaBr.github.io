Analisador Léxico

Linguagem: C

Repositório: GitHub

Projeto desenvolvido como requerimento para a disciplina de Compiladores, cursada no 2º Quadrimestre de 2018. Trata-se de um analisador léxico para uma variação da linguagem "javalete", conforme especificação proposta.

A análise léxica é a primeira parte do processo de compilação. O trabalho do analisador é interpretar o conjunto de caracteres digitados no arquivo fonte e transformá-lo em tokens que serão utilizados nas próximas etapas de compilação.

Nessa etapa não se preocupa em saber se as palavras digitadas respeitam a sintaxe correta, apenas se elas estão escritas da forma correta. Dessa forma são identificados apenas os erros mais simples relacionados ao alfabeto utilizado, ou seja, se houve a utilização de caracteres inválidos em determinadas palavras.

A implementação foi realizada através de uma máquina de estados que avaliava os caracteres da entrada um a um. Assim que havia um espaço, se a máquina estivesse num estado de aceitação, gerava o token correspondente à palavra lida. Caso a máquina não estivesse num estado de aceitação, retornava um token de erro. Note que a máquina ignorava comentários e espaços em branco consecutivos, reduzindo o processamento necessário.

O projeto foi realizado em conjunto com mais dois alunos, porém fui o principal responsável pelo código, enquanto os outros se concentraram mais no planejamento da máquina de estados e a redação do artigo que foi entregue como parte dos requisitos da disciplina.

Analisador Sintático

Linguagem: C

Repositório: Analisador Sintatico

Esse foi o 2º Projeto desenvolvido para a disciplina de Compiladores, cursada no 2º Quadrimestre de 2018. Esse analisador sintático foi desenvolvido para analisar tokens emitidos pelo analisador léxico anterior, segundo as regras da linguagem "javalete", conforme a especificação.

A análise sintática é a segunda parte do processo de compilação. O trabalho do analisador é verificar se os tokens gerados pelo analisador lexico obedecem a gramática correta. Dessa forma, não basta que as palavras estejam corretas, mas elas devem estar numa ordem correta.

BlackJack Funcional

Linguagem: Lisp - Dialeto Racket

Repositório: Analisador Sintatico

Esse projeto foi realizado para a disciplina de Paradigmas da Programação, cursada no 2º Quadrimestre de 2018. Trata-se de um jogo de BlackJack(21) em que o jogador se conecta a um servidor via TCP, e joga uma partida de 21 contra o dealer. O principal requisito do projeto era a abordagem Funcional do problema. Nosso grupo inicialmente foi composto por quatro elementos, mas foi concluído com apenas três.

O jogo de BlackJack é bem simples, o jogador e o dealer se revezam comprando cartas até que queiram parar, ou que sua pontuação estoure 21.

A implementação desse projeto, realizada em linguagem até então desconhecida por nós, envolveu:

  • Comunicação via TCP-IP: Para que o cliente se conectasse ao servidor.
  • Criação de uma interface para que o cliente interagisse com o servidor.
  • Criação de uma inteligência artificial baseada em máquina de estados para jogar contra o jogador
  • Uso de funções e chamadas flexíveis o suficientes para que fosse possível ter um jogo com mais jogadores interagindo caso houvesse tempo hábil.

Originalmente o trabalho seria sobre o jogo de Pife-Pafe, e nosso grupo se dividiu para que fizessemos a comunicação via sockets, a inteligência artificial, a interface com o jogador e a modelagem das regras do jogo. O relatório faríamos juntos Infelizmente, durante o quadrimestre um dos alunos não pôde mais participar da matéria. Dessa forma, tivemos que nos replanejar. Redistribuimos as tarefas e realizamos encontros onlines para que trabalhassemos juntos no projeto. Como ainda assim percebemos que não seríamos capazes de entregar o jogo de Pife-Pafe completo, e como o tipo de jogo não era relevante dados os requisitos da matéria, conversamos com o professor e alteramos o jogo para um mais simples, o BlackJack. Dessa forma foi possível entregar um trabalho completo, escrito utilizando o paradigma funcional.
